# ECE Organization Website

This is the official website of the Electronics Engineering (ECE) Student Organization. It is a simple, responsive website built using HTML and CSS, and showcases our accomplishments, announcements, ECE background information, downloadable e-books, and profiles of current and past officers.

## 📌 Features
- 🏆 Accomplishments Page
- 📢 Announcements Section
- 📚 Downloadable E-books
- 👨‍💼 Current and Past Officers Profile
- 📖 ECE Program Background
- 📬 Contact Form

## 🛠️ Tech Stack
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (planned for interactivity)

## 🚀 How to Run Locally
1. Download or clone this repository.
2. Open `index.html` in your web browser.
3. Edit using any code editor (VS Code recommended).
